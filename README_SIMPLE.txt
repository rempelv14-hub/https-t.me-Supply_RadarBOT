SUPPLIER RADAR — ПРОСТАЯ ИНСТРУКЦИЯ

Что делает бот:
1) Сам ищет публичные Telegram-группы по теме поставщиков, баз, косметики, WB/Ozon, закупа из Европы.
2) Может сам вступать в публичные группы.
3) Ищет внутри групп сообщения от людей, которым нужны поставщики.
4) Отсекает рекламу и продавцов баз.
5) Отправляет найденные лиды админу в Telegram-бота.
6) Если Telegram дает FloodWait, бот предупреждает админа, ждет и сам запускает поиск снова.

==================================================
ШАГ 1. ВСТАВЬ ДАННЫЕ В config.py
==================================================

Открой файл config.py и заполни:

BOT_TOKEN = "токен из BotFather"
ADMIN_IDS = [твой Telegram ID]
API_ID = твой api_id
API_HASH = "твой api_hash"

SESSION_STRING пока оставь пустым:
SESSION_STRING = ""

Telegram ID можно узнать у @userinfobot

API_ID и API_HASH тут:
https://my.telegram.org/apps

==================================================
ШАГ 2. УСТАНОВИ ЗАВИСИМОСТИ
==================================================

Вариант простой:
Дважды нажми файл:

00_SETUP.cmd

Если окно закрывается, сделай так:
1) Открой папку проекта.
2) В строке пути сверху напиши cmd и нажми Enter.
3) В черном окне введи:

00_SETUP.cmd

==================================================
ШАГ 3. ПОЛУЧИ SESSION_STRING
==================================================

Запусти:

01_GET_SESSION.cmd

Он попросит:
- номер телефона Telegram
- код из Telegram
- пароль 2FA, если он есть

После этого появится длинная строка SESSION_STRING.

Скопируй ее полностью и вставь в config.py:

SESSION_STRING = "сюда_длинную_строку"

Важно: строка должна быть в кавычках и без переносов.

==================================================
ШАГ 4. ЗАПУСТИ БОТА
==================================================

Запусти:

02_RUN_BOT.cmd

После запуска бот:
- подключит Telegram session
- просканирует уже подключенные группы
- через 15 секунд начнет автопоиск новых групп
- при FloodWait предупредит админа и сам продолжит позже

==================================================
КОМАНДЫ В БОТЕ
==================================================

/start — запуск
/id — узнать свой Telegram ID
/stats — статистика
/leads — последние лиды
/groups — найденные группы
/discovery — статус автопоиска групп
/findgroups — запустить поиск групп вручную
/keywords — ключевые слова
/help — помощь

==================================================
ВАЖНО
==================================================

1) Если бот уже запущен на Railway, не запускай этот же BOT_TOKEN на компьютере одновременно.
Иначе будет ошибка getUpdates conflict.

2) Бот не может входить в закрытые private-группы без invite-ссылки.
Сам он может искать и вступать только в публичные группы.

3) FloodWait — это лимит Telegram.
Если он появится, бот напишет админу, сколько ждать.
После окончания лимита бот сам запустит новый поиск групп.

4) Для лучшего результата сначала вручную добавь Telegram-аккаунт в 10-20 хороших тематических групп.
Потом автопоиск будет расширять базу.
